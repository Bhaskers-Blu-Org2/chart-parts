Class SceneFrame

The scene frame is analagous to a stack-frame - it contains contextual information for a section of a scene, including dimensions, scales, data, and event handlers

pushView

  • pushView(view: ViewSize, viewTL?: [number, number], viewBR?: [number, number]): SceneFrame
    • Defined in scene/SceneFrame.ts:95

    Pushes a new sceneframe with an updated viewspace. Recomputes scales.

    Parameters

    • view: ViewSize

      The new view to push. This is the size of the group that will be inserted

    • Default value viewTL: [number, number] = [0, 0]

      The top-left of the draw-area within the view. This does not affect the emitted group, but does affect draw-scales

    • Default value viewBR: [number, number] = [view.height, view.width]

      The bottom-right of the draw-area within the view. This does not affect the emitted group, but does affect draw-scales
